What is Full-Stack Development?


Last Updated: 30 Apr, 2021

Full stack development: It refers to the development of both front end(client side) and back end(server side) portions of web application.


What is a Full-Stack Developer?


Last Updated: 30 Apr, 2021

Full stack web Developers: Full stack web developers have the ability to design complete web application and websites. They work on the frontend, backend, database and debugging of web application or websites.

Some may wonder, what exactly is a Stack?


Last Updated: 30 Apr, 2021

Stack: A stack is simply a collection of software required for the production of web applications or websites. There are several parts to a web development stacks including the programming languages used, web frameworks, database, web servers, etc.

These include:

  • MEAN Stack: MongoDB, Express, AngularJS and Node.js.
  • MERN Stack: MongoDB, Express, ReactJS and Node.js
  • Django Stack: Django, python and MySQL as Database.
  • Rails or Ruby on Rails: Uses Ruby, PHP and MySQL.
  • LAMP Stack: Linux, Apache, MySQL and PHP.
