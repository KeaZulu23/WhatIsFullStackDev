What is PHP?


Last Updated: 30 Apr, 2021

PHP: PHP is a server-side scripting language designed specifically for web development. PHP can be easily embedded in HTML files and HTML codes can also be written in a PHP file. The thing that differentiates PHP from a client-side language like HTML is, PHP codes are executed on the server whereas HTML codes are directly rendered on the browser.


Why is it so popular?


Last Updated: 30 Apr, 2021

Popularity: First, PHP is open source. What this means is that the actual programming used for PHP scripts is open for everyone to see. Combined with PHP's popularity, this ensures that there are a lot of people working on improving it. This means that PHP is also very inexpensive and very reliable.

PHP can also be used in many platforms and is compatible with many servers. PHP scripts may be executed on a computer that runs on Windows and will have no problems when it is run on a Unix or Linux machine.

Last but not least, PHP is very flexible.

So what is it's purpose?


Last Updated: 30 Apr, 2021

Purpose: While PHP language is universal overall, it is best suited for web development. There are multiple ways of using PHP when it comes to creating web applications.

Well, you can:

  • Create dynamic content.
  • Create, open, write, read, delete, and close files on the server.
  • Collect data from forms.
  • Add and modify database information.
  • Perform data encryption.
  • Receive and send cookies.
